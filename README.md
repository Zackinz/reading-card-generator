## Usage

### Browser

Open `index.html` in your browser.

**Browser compatibility**

See [html2canvas](https://github.com/niklasvh/html2canvas#browser-compatibility).

### Electron

```js
$ npm install
$ npm start
```

## Build

```bash
// for macOS
$ npm run dist-mac

// for windows x64
$ npm run dist-win
```
